# Security Policy

## Supported Versions

Only the latest stable version receives security updates.

---

## Reporting a Vulnerability

If you discover a security issue, please do not publish it immediately.

Instead:

Email:

sedsist@gmail.com

Include:

- Description
- Steps to reproduce
- Operating System
- Version
- Possible impact

We will investigate as quickly as possible.

---

## Security Goals

SISDIGEM Document Studio is designed to:

- Never upload user documents without permission.
- Process files locally whenever possible.
- Avoid unnecessary network communication.
- Respect user privacy.
- Keep dependencies updated.

---

## Responsible Disclosure

Researchers who report vulnerabilities responsibly will be acknowledged.

Thank you for helping improve project security.
